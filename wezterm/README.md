# WezTerm Configuration

This directory contains the WezTerm terminal emulator configuration.

## Files

- `.wezterm.lua` - Main WezTerm configuration file

## Installation

The configuration will be symlinked to `~/.wezterm.lua` by the install script.

## Customization

Edit `.wezterm.lua` to customize:
- Font settings
- Color schemes
- Window geometry
- Keybindings
- And more

Refer to the [WezTerm documentation](https://wezfurlong.org/wezterm/) for all available options.
